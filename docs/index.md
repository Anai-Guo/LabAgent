# LabAgent

**Your AI-powered lab assistant.** From research question to publication-ready data.

## What is LabAgent?

LabAgent is an open-source framework that connects AI to laboratory instruments. It supports 46 measurement templates across 9 scientific disciplines, with built-in safety boundaries and experiment memory.

## Quick Install

```bash
pip install git+https://github.com/Anai-Guo/LabAgent.git
```

## Key Features

- **46 measurement templates** across 9 disciplines
- **AI at every step** — classification, planning, analysis, interpretation
- **3-tier safety system** — block / confirm / allow
- **6 AI model providers** — Claude, GPT, Gemini, Ollama, vLLM, DeepSeek
- **Adaptive Web GUI** with real-time monitoring
- **Terminal panel** (Claude Code-style)
- **MCP server** for AI IDE integration
