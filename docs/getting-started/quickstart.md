# Quick Start

## 1. Scan Your Instruments
```bash
labharness scan
```

## 2. Generate a Measurement Plan
```bash
labharness propose hall
```

## 3. Launch the Web GUI
```bash
labharness web
```
Open http://localhost:8080 in your browser.

## 4. Interactive AI Chat
```bash
labharness chat
```
Or use the terminal panel:
```bash
pip install "lab-agent[tui]"
labharness panel
```
